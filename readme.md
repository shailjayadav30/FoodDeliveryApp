# 🍔 Food Delivery App

A modern and responsive Food Delivery Mobile App built using React Native and Expo.
The app provides a clean UI for browsing restaurants, exploring menus, adding food items to cart, and viewing restaurant details.

---

# ✨ Features

- Browse restaurants
- View restaurant details
- Explore food categories
- Add items to cart
- Dynamic cart updates
- Beautiful food cards UI
- Horizontal drinks slider
- Responsive layouts
- Reusable components
- Smooth navigation

---

# 🚀 How To Run Locally

## 1. Clone the repository

```bash
git clone https://github.com/your-username/food-delivery-app.git
```

## 2. Navigate into project

```bash
cd food-delivery-app
```

## 3. Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

## 4. Start Expo server

```bash
npx expo start
```
## 5. Run on device

* Scan QR code using Expo Go app
